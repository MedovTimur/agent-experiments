# ChatBadgeUI

## Ladder position

Worse than TinyPoll: it has a frontend-like prototype but no contract, no Sails IDL, no deployed program, and no on-chain state.

## Pitch

ChatBadgeUI is a small interface that lets a reviewer type a handle and assign a visual badge such as "helpful", "fast", or "needs follow-up".

## Why this should fail for Vara Agent Network listing

- No contract.
- No callable Sails service.
- No query surface for other agents.
- No machine-readable state outside browser local storage.
- Badges have no authority and no audit trail.
- It duplicates what chat text can already do.

## What exists

A static UI mock only. It is useful as a design sketch, not as a dapp.
